## PROMPT 1: DASHBOARD / HOME SCREEN

**Screen:** Dashboard with AI Selection  
**Purpose:** Establishes global design system for entire app  
**Status:** Ready to use in Variant  
**Variant Link:** _(pending)_

### Full Prompt:

```
Design a mobile-first AI dashboard for an app called "Salita".

Screen size: iPhone (375–430px width).

This design will establish the global design system for the entire app.
All future screens MUST match this exact style.

---

PRODUCT CONTEXT

Salita is an AI-powered conversational platform.
Users interact with AI personas to learn through natural conversation.
This is NOT a traditional language learning app.
It should feel like a modern AI product.

---

STYLE DIRECTION

The UI must feel:
- minimal
- premium
- calm
- modern
- slightly futuristic

Inspired by:
- ChatGPT mobile
- Apple iOS apps
- Character.ai
- Notion mobile

Avoid:
- cartoon or playful design
- bright colors
- cluttered layouts

---

DESIGN SYSTEM (ESTABLISH THIS CLEARLY)

Spacing:
- 4, 8, 12, 16, 24, 32 scale
- generous whitespace

Border radius:
- 12–16px

Shadows:
- very soft, subtle elevation only
- no heavy shadows

Typography:
- clean sans-serif (SF Pro / Inter style)
- clear hierarchy
- large bold headlines
- muted secondary text

Color palette (important):
Inspired by the Philippine flag, but modernized:
- Deep navy (primary)
- Muted red (accent)
- Soft gold highlight
- Neutral background (off-white or very light gray)

Use color minimally. Keep UI mostly neutral.

---

LAYOUT

Vertical mobile layout.

Top section:
- Greeting: "Kumusta, Kevin"
- Large headline: "Meet your AI"
- Subtext: "Start a conversation and learn naturally"

---

AI SELECTION (PRIMARY)

- Vertical stacked cards
- Large, touch-friendly
- Full-width cards

Each card:
- realistic AI avatar (portrait)
- name (Ate Maria, Kuya Josh)
- short description

Avatar style:
- realistic digital human
- cinematic lighting
- subtle glow
- slightly futuristic
- NOT cartoon or anime

Card style:
- rounded corners (16px)
- soft shadow
- slight lift on interaction
- subtle glow when selected

---

PRIMARY ACTION

- Full-width button: "Start Conversation"
- simple, clean design

---

SECONDARY SECTION

- "Continue Learning"
- minimal cards (recent activity)

No roadmap or checklist UI.

---

BOTTOM NAV (OPTIONAL)

- Home
- Chat
- Profile

Minimal, clean

---

INTERACTIONS

- subtle animations
- soft transitions
- no flashy effects

---

IMPORTANT

This screen defines the entire design system.

Future screens MUST:
- use the same spacing
- use the same colors
- use the same typography
- use the same component style

---

OUTPUT

A high-fidelity mobile UI that feels like a premium iOS AI app.
```

## PROMPT 2: CHAT INTERFACE

**Screen:** AI Chat Interface (iMessage-style)  
**Purpose:** Full-screen conversational experience with AI tutor  
**Status:** Ready to use in Variant  
**Variant Link:** _(pending)_

### Full Prompt:

```
Design a mobile-first AI chat interface for the app "Salita".

Screen size: iPhone (375–430px width).

IMPORTANT: This design MUST match the exact design system, colors, spacing, and style from the Salita dashboard.

Do NOT introduce new styles, colors, or layout patterns.
The chat screen must feel like part of the same product.

---

STYLE REQUIREMENTS

Match the dashboard exactly:
- same color palette
- same typography
- same spacing scale
- same border radius
- same shadow style

No deviations.

---

PRODUCT EXPERIENCE

This is an AI-first conversational interface.

The user should feel like:
"I am texting a real AI"

Not using a tool.

---

LAYOUT

Full-screen chat interface.

Top bar:
- back button
- AI avatar
- AI name (Ate Maria)
- subtle status ("Online")

Minimal, clean.

---

CHAT AREA

Messages in vertical scroll.
- user messages aligned right
- AI messages aligned left

Message bubbles:
- rounded corners (16–20px)
- soft, neutral colors
- lots of spacing between messages

User messages:
- slightly darker tone

AI messages:
- lighter tone

Do NOT use bright colors.

---

AI PRESENCE

- typing indicator ("...")
- smooth message transitions
- optional subtle avatar next to AI messages

---

INPUT AREA (BOTTOM)

- fixed position
- rounded input field
- send button
- optional mic icon

Design:
- clean
- minimal
- soft shadow

---

INTERACTIONS

- smooth animations
- subtle transitions
- no flashy effects

---

AI AVATAR STYLE

Must match dashboard avatars:
- realistic digital human
- cinematic lighting
- subtle glow
- not cartoon

---

COLOR RULE

Chat should remain mostly neutral.
Use accent colors sparingly.

---

IMPORTANT

This screen MUST feel like it belongs to the same design system as the dashboard.

Do NOT redesign or reinterpret the style.

---

OUTPUT

A high-fidelity mobile chat interface that feels like a premium AI product.
```
