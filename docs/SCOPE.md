## THE PROBLEM

~4 million Filipino Americans live in the US. Second-generation Filipino Americans are losing Tagalog at an alarming rate. There is no high-quality app that helps them reconnect with the language through real conversation and cultural context.

Existing language apps (Duolingo, Babbel, Rosetta Stone):
- Do not support Tagalog well or at all
- Teach like textbooks (flashcards, drills, boring)
- Do not reflect real Filipino conversation (family phrases, slang, Taglish)

---

## THE SOLUTION

Salita is an AI conversation partner that teaches Tagalog through real conversation.

Core idea: **"Learn Tagalog by talking."**

There are:
- No lessons
- No flashcards
- No quizzes

Users learn by chatting with a Filipino AI tutor that:
- speaks naturally
- guides them on what to say
- corrects gently
- explains cultural context

---

## CORE PRODUCT PRINCIPLES (NON-NEGOTIABLE)

1. Conversation-first (chat is the entire product)
2. User MUST speak Tagalog immediately
3. AI must guide the user (never leave them stuck)
4. No traditional learning systems (no lessons, quizzes, dashboards)
5. Must feel like texting a real Filipino person
6. Must be fast (<2 second response target)
7. AI output must be structured and consistent

---

## CORE USER EXPERIENCE (MVP FLOW)

1. User signs up
2. User selects a tutor persona
3. User is dropped directly into chat
4. Tutor begins conversational onboarding
5. User is guided to respond in Tagalog
6. User learns through back-and-forth conversation
7. User can share moments
8. System also generates influencer content from tutors

---

## 4. GUIDED LEARNING SYSTEM (KEY DIFFERENTIATOR)

This is the most important system.

The AI must teach through guidance, not explanation.

---

### Every AI message MUST include:

1. Tagalog message (primary)
2. English translation (secondary)
3. Hint (how to respond)
4. Example responses (buttons)
5. Optional correction
6. Optional cultural note

---

### Example

**Tagalog:** "Ano ang pangalan mo?"  
**English:** "What is your name?"  
**Hint:** "Sabihin: 'Ako si ___'"  
**Examples:**
- "Ako si Kevin"
- "Ako si Maria"

---

### Requirements

- User must always know what to say
- No blank states
- No confusion
- Tagalog is always primary
